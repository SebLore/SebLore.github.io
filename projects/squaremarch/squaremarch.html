This is my implementation of marching squares. It's the result of a course I did in WebGL where the task was to implement Metaballs in WebGL. The instructions were to 'just do it', so I did my own reasearch and found that there are generally two ways to do it. The first is to use raymarching and the second is to use marching squares. I chose to use marching squares because I am more familiar with rasterization than ray-based rendering.
