Bubble Sort

Bubble Sort is an easy sorting algorithm to understand, and is usually the first one taught in computer science courses. It works by repeatedly stepping through an array (a list of values like numbers), comparing adjacent elements and swapping them if they are in the wrong order. The pass through the list is repeated until no swaps are needed, which means the list is sorted.
