Bubble Sort

What is bubble sort

Bubble sort is a simple sorting algorithm that checks two values against each other and swaps their positions in the array if the first is larger than the second. It functions by picking the first element in an array, then iterating across the array until it finds an element that is bigger than it, at which point that element will be replace the starting element for this iteration.

This program is heavily inspired by the work of Timo Bingmann, specifically the project "The Sound of Sorting". I wanted to do my own implementation, partially as practice in JavaScript, partially to have something to put on my website.
